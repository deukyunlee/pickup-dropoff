# pickup-dropoff

본 프로젝트는 유치원 등하원 시스템의 문제점을 해결하고자 제작되었습니다.

기능으로는 명단작성, 출결확인, 이름 및 반 조회, 출결 작성 등이 있습니다.

(사진은 프로젝트 완성 시 추가예정)

이름과 수업 분반, 차량 분반을 선택하면 DB에 정보가 입력됩니다.
